# Academic-Only Boundary Declaration

This repository is an academic mirror of the Spectrum project.

It exists solely for:
- Formal definitions
- Mathematical proofs
- Theoretical models
- Explanatory documents
- Diagrams and derivations

## Prohibited Content
The following MUST NOT exist in this repository:
- Executable production code
- Runtime logic
- Deployment artifacts
- Benchmarks tied to production execution
- Optimizations intended for implementation

All executable logic lives exclusively in the `spectrum` repository.

Violations of this boundary invalidate the purpose of this repository.
